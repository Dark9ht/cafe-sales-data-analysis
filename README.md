# Cafe Sales Data Analysis

## Project Overview
This project analyzes a cafe sales dataset to explore customer purchasing behavior, payment preferences, and product performance.

The dataset contained missing values and inconsistent formatting, which required data cleaning before analysis.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Data Cleaning
The dataset required several cleaning steps:

- Removed extra spaces and standardized text formatting
- Converted numeric columns (Quantity, Price, Total Spent)
- Converted transaction date to datetime format
- Handled missing values

## Business Questions
1. What items generate the highest sales?
2. What payment methods are most commonly used?
3. Do customers prefer in-store orders or takeaway?

## Key Findings
- Some menu items generate significantly higher revenue than others.
- Payment methods are used at relatively similar rates.
- Customers use both takeaway and in-store options frequently.

## Conclusion
The analysis provides insights into product performance and customer purchasing behavior. The results highlight the importance of maintaining multiple payment options and balanced service between takeaway and in-store customers.
